CSS Positioning Reference Table

Tag Name Description
position: relative Places an element realtive to its current position without changing the layout around it
position: absolute Places an element relative to its parent's position and changing the layout around it
float: left CSS Positioning Floats an element to the left side of its container. The width of the container must be specified or the element will assume the full width of its containing element.
float: right CSS Positioning Floats an element to the right side of its container. The width of the container must be specified or the element will assume the full width of its containing element.
